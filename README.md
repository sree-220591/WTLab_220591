# Hands in Need — Disaster Management Portal

A static website (HTML/CSS/JS) created as a web technologies lab project.

## Structure
- Task01_Disaster_Management_HTML/ — HTML pages (index.html, about.html, services.html, contact.html)
- Task02_Disaster_Management_CSS/ — CSS stylesheets
- script.js — site JavaScript (carousel, UI behaviors)

## Local preview
Open `Task02_Disaster_Management_HTML/index.html` in a browser, or run a simple HTTP server:

- Python 3:
  ```bash
  python -m http.server 8000
  ```
  Then open: http://localhost:8000/Task01_Disaster_Management_HTML/index.html

## Suggestions / Next steps
